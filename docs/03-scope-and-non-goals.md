# Scope and Non-Goals

## In scope for v1
- support comprehensive income tax preparation workflows for agent-capable users
- ingest filing-relevant revenue, expense, and document data from mixed sources
- maintain a review queue for ambiguous items
- generate a structured filing draft
- assist with HomeTax navigation/input where feasible
- keep an audit trail of automated decisions

## Explicit non-goals for v1
- silent, fully autonomous filing without user visibility
- replacing certified tax or legal professionals
- supporting every taxpayer segment equally on day one
- building a regulated financial-data aggregation business
- guaranteeing tax outcomes from incomplete or low-quality source data

## Initial user profile
- comfortable using AGENT AI or agent-assisted software
- willing to self-host or run a semi-technical setup
- able to complete consent and authentication flows when prompted
