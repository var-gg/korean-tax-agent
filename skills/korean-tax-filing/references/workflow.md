# Workflow

1. Confirm setup state.
2. Explain available source connections.
3. Ask for required consent only for chosen sources.
4. Import and normalize data.
5. Run classification.
6. Generate a review queue.
7. Resolve review items in batches when safe.
8. Compute filing draft.
9. Summarize warnings and assumptions.
10. Start HomeTax assistance only after the draft is ready.
11. Stop for explicit final submission approval.
